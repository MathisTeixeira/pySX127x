# Install

- Install Raspberry Pi OS (64 bits)
- sudo apt update
- sudo apt upgrade
- pip install opencv-python==4.5.3.56
- sudo raspi-config nonint do_spi 0
- pip install RPi.GPIO
- pip install spidev
- pip install pyLoRa
- pip install pycryptodome
- pip install pycrypto
